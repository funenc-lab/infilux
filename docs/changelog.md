# Changelog

## v0.4.3 - 2026-04-27

### Added

- Added Skill and MCP policy controls for supported agent sessions, including global, repository, worktree, and per-session launch scopes.
- Added a Claude capability catalog for discovered skills, commands, subagents, shared MCP servers, and personal MCP servers.
- Added OpenCode to the built-in agent list.
- Added agent startup diagnostics to make session launch failures easier to inspect.

### Fixed

- Hardened capability policy runtime control for provider-specific launch behavior.
- Added recoverable error states for deferred file panels when lazy loading fails.

### Changed

- Running agent sessions are now marked stale when Skill or MCP policy changes require a restart.
- Updated Agent session controls and action icons around Skill and MCP launch options.

### Notes

- GitHub Release assets and downloadable package links are generated by the release workflow after the `v0.4.3` tag is pushed.

## v0.4.2 - 2026-04-23

### Fixed

- Normalized Codex subagent cwd filters by path shape so macOS-style workspace keys still match on non-macOS hosts.

### Changed

- Added a regression test to keep Linux-style workspace filters case-sensitive while preserving darwin and windows normalization behavior.

### Notes

- GitHub Release assets and downloadable package links are generated by the release workflow after the `v0.4.2` tag is pushed.

## v0.4.1 - 2026-04-23

### Added

- Added a dedicated session subagent trigger policy so pending sessions can keep the trigger visible without over-emphasizing it.

### Fixed

- Restored chat canvas subagent inspector scrolling behavior.
- Stabilized main-process shutdown cleanup to reduce teardown-related failures.

### Changed

- Updated the session subagent trigger to use a branch-oriented icon and explicit emphasis state.
- Expanded renderer coverage around session subagent trigger presentation and inspector wiring.

### Notes

- GitHub Release assets and downloadable package links are generated by the release workflow after the `v0.4.1` tag is pushed.

## v0.4.0 - 2026-04-22

### Added

- Added a floating session subagent inspector and a dedicated session subagent inspector view.
- Added chat-canvas focus routing for notification-driven agent sessions.
- Added a setting to keep session-backed chat panels mounted after their idle cooldown expires.

### Fixed

- Reduced renderer-side worktree and file tree retention leaks.
- Stabilized agent canvas viewport behavior during panel activity changes.
- Ensured signal-driven shutdown paths exit reliably.
- Kept newly created worktrees visible while the agent activity filter is active.
- Surfaced tmux host resource exhaustion earlier during session startup checks.

### Changed

- Isolated editor runtime policies to tighten panel lifecycle boundaries.

### Notes

- GitHub Release assets and downloadable package links are generated by the release workflow after the `v0.4.0` tag is pushed.
